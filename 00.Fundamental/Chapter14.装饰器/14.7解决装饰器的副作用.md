# 解决装饰器改名的副作用

- 在装饰器闭包函数头顶，添加语法糖 **@wraps**

- @wrpas 将 @wraps(func) 中的 func 函数 参数的一系列的信息复制到 wrapper 闭包函数上面
  
```python
from functools import wraps

def decorator(func):
    @wraps(func)
    def wrapper():
    pass
    return wrapper
```

```python
import time
from functools import wraps

def decorator(func):
    @wraps(func)
    def wrapper():
        print(time.time())
        func()
    return wrapper

@decorator
def f1():
    '''
        This is f1
    '''
    print(f1.__name__)

print(help(f1))

f1()

# Help on function f1 in module __main__:

# f1()
#     This is f1

# None
# 1584995219.4156177
# f1
```
