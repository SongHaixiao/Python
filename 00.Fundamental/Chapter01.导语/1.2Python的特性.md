# Python 的特性

### Python 与 Python的特点
1. Python 是一门编程语言，它只是众多编程语言中的一
   
   Python 是语言，不是框架。

   框架是以编程语言构建的一系列基础功能的集合 

2. 语法简洁、优雅、编写的程序容易阅读

3. 跨平台，可以运行在Windows、Linux以及MacOS

4. 易于学习。站在非计算机专业的角度来讲，如果把编程语言当作解决问题的工具，Python确实相较于C++、Java、JavaScript等语言要易于学习和掌握。

5. 极为强大而丰富的标准库与第三方库，比如电子邮件，比如GUI界面

6. Python 是面向对象的语言
