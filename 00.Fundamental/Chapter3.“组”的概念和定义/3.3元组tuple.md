# 元组
 
- **<font color="red">（value1,value2,...）</font>** 


- 元组 **tuple** 操作和列表 **list** 操作相似

>
    
    >>> （1，2，3）
    （1， 2，3，）
    >>> (1,'1',True)
    (1,'1',True)
    >>> (1,2,3,4)[0]
    1
    >>> (1,2,3) + (4,5,6)
    (1,2,3,4,5,6)
    >>> (1,2,3) * 3
    (1,2,3,1,2,3,1,2,3)

- () 表示 算数优先级 和 表示元组 会发生冲突
    - 当 () 内，只有一个元素时，表示算数优先级
    - 当 () 内，不只有一个元素时，表示元组
>
    
    >>> type((1))
    <class 'int'>
    >>> type(1, )
    <class 'tuple'>
    >>> type(1, 2, 3)
    <calss 'tuple'>
