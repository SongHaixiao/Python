# 数据科学领域5个最佳 Python 库

Numpy / Scipy / Pandas / Matplotlib / Scikit-learn

### **Numpy**

- N 维数组（矩阵），快速高效，矢量数学运算
  
- 高效的 Index，不需要循环

- 开源免费跨平台，运行效率足以和 C/Matlab 媲美

![Numpy](Resource/10.png)

### **Scipy**

- 依赖于 Numpy

- 专为科学和工程设计

- 实现了多种常用科学计算 ： 线性代数、傅里叶变换、信号和图像处理
  
### **Pandas**

- 结构化数据分析利器 （依赖 Numpy）

- 提供了多种高级数据结构 ： Time-Series， DataFrame， Panel

- 强大的数据索引和处理能力
