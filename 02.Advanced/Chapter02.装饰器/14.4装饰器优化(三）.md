# 装饰器优化（三）

- 使装饰器可以传入 **<font color="red">关键字参数</font>**

- 关键字参数 **\*\*kw**，与 可变参数 **\args**一样，可以改变名字，但是一般情况下，没有应用逻辑时，就用两个名字
  
```python
import time

def decorator(func):
    def wrapper(*args, **kw):
        print(time.time())
        func(*args, **kw)
    return wrapper

# 引用调用
@decorator
def f1(func1):
    print('This is a function named ' + func1)

# 直接调用
@decorator
def f2(func2_1, func2_2):
    print('This is a function named ' + func2_1)
    print('This is a function named ' + func2_2)




@ decorator
def f3(func_name1, func_name2, **kw):
    print('This is a function named ' + func_name1)
    print('This is a function named ' + func_name2)
    print(kw)


f1('func1')
f2('func2_1','func2_2')
f3('func3_1', 'func3_2', a = 1, b = 2, c = '123')2

# 1584904049.468628
# This is a function named func3_1
# This is a function named func3_2
# {'a': 1, 'b': 2, 'c': '123'}
```
