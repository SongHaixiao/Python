# 初识正则表达式

- 正则表达式是一个特殊的字符序列,来判断一个字符串是否与我们所设定的这样字符序列相匹配

## 作用
- 用来快速检测文本、实现一些替换文本的操作

- 例如：
  1. 检查一串数字是否是电话号码
  2. 检测一个字符串是否符合 email
  3. 把一个文本里指定的单词替换为另一个单词
   

## 语法

```python
import re
re.findall(pattern, string,flags=0)
```
