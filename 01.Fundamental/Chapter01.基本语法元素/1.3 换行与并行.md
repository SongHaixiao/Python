# 换行与并行

在Python程序编写过程中，有时会遇到两行代码放在同一行更易懂或者一行中过长的代码为了结构清晰易懂不适合放到同一行中。

下面将探讨在Python中如何处理代码换行与并行的问题。

## 1. 代码换行

在Python编程中一般是一行写完所有代码，如果遇到一行写不完需要换行的情况，也允许采用代码换行的方式将一行代码分成多行编写。

有如下4种方法供选择：

### （1）在该行代码末尾加上续行符 `"\"`:

```python
print("2020\
    我在学习\
    Python!")

# Output
# 2020    我在学习    Python!
```

### (2) 采用单引号：

```python
print('2020'
    '我在学习'
    'Python!')

# Outpu
# 2020我在学习Python!
```

### (3) 采用三个双引号：

```python
print("""2020"""
    """我在学习"""
    """Python!""")

# Output
# 2020我在学习Python!
```
