# 数学基础回顾之矩阵运算

### 矩阵

- 矩阵 ： 矩形的数组，即二维数组。其中向量和标量都是矩阵
的特列.

- 向量 ：是指 1 x n 或者 n x 1 的矩阵.

- 标量 ：1 x 1 的矩阵.

- 数组 ：N 维的数组，是矩阵的延伸.
